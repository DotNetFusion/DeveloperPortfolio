#  Sunkolu Shiva Kumar - Developer Portfolio

A modern, responsive, and professionally designed developer portfolio website built with **Angular 21**, **TypeScript**, and **SCSS**. Featuring smooth animations, dark theme, and production-ready code.

##  Features

###  Sections Included

- **Hero Section** - Eye-catching introduction with animated gradient text and profile image
- **About Section** - Professional summary with highlight cards and statistics
- **Skills Section** - Organized technical skills by category with emoji badges
- **Projects Section** - Showcase of featured projects with GitHub and live demo links
- **Experience Section** - Professional timeline with company details and responsibilities
- **Certifications Section** - Professional certifications and achievements
- **GitHub Section** - GitHub activity and code snippet display
- **Contact Section** - Functional contact form with validation
- **Footer Section** - Quick navigation and social media links

###  Technical Features

-  **Angular 21** - Latest stable version with standalone components
-  **Standalone Components** - No NgModules, modern Angular approach
-  **TypeScript** - Fully typed for better developer experience
-  **SCSS** - Professional styling with variables, mixins, and animations
-  **Responsive Design** - Mobile-first approach, optimized for all devices
-  **OnPush Change Detection** - Optimal performance and efficiency
-  **Angular Signals** - Modern state management approach
-  **Reactive Forms** - Validated contact form with error handling
-  **Server-Side Rendering (SSR)** - Better SEO and initial load time
-  **CSS Animations** - Smooth, performant animations throughout
-  **Accessibility** - WCAG compliant with ARIA labels
-  **Dark Theme** - Modern, professional dark color scheme
-  **SEO Optimized** - Meta tags, structured data, semantic HTML

##  Prerequisites

- Node.js 18.x or higher
- npm 11.x or higher
- Git

##  Quick Start

### 1. Clone Repository
\\\ash
git clone https://github.com/DotNetFusion/DeveloperPortfolio.git
cd DeveloperPortfolio
\\\

### 2. Install Dependencies
\\\ash
npm install
\\\

### 3. Run Development Server
\\\ash
npm start
\\\

### 4. Build for Production
\\\ash
npm run build
\\\
